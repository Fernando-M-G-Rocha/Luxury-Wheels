\## 🚗 Luxury Wheels



\*\*Fleet Management System\*\* developed with \*\*Python\*\*, \*\*Tkinter\*\*, \*\*Flask\*\* and \*\*SQLite\*\*.



Luxury Wheels is a desktop and web application designed to manage vehicle fleets, customers, reservations and payments through an intuitive interface.



The project was developed as a personal portfolio project to strengthen my software development skills and demonstrate practical knowledge in Python application development.



\## ✨ Features

\- Dashboard with statistics

\- Vehicle management

\- Customer management

\- Reservation management

\- Payment management

\- Vehicle availability tracking

\- Maintenance alerts

\- Inspection reminders

\- Excel export

\- Authentication system



\## 🛠 Technologies

\- Python

\- Tkinter

\- SQLite

\- Pandas

\- OpenPyXL

\- Flask (Web version)
